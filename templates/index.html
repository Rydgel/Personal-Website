About this website

This site was made on a Mac, designed in browser and Photoshop (for the fancy logos).

It is powered by Python and the Flask micro-framework. Edited with TextMate 2. The source code is hosted on Github, take a look if you want to.

Hosting

Everything is hosted painlessly on a cloud platform called Heroku (pronounced her-OH-koo). It’s a fun way of building and deploying web apps.

Fonts

All fonts are served up by the wonderful Typekit. Body copy is the awesome FF Quadraat Web by Fred Smeijers.
